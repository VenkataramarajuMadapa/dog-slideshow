This is my implementation of Brad Schiff's **Dogs, JavaScript & An API** project.

## Description
- Fetches dog breeds using the Dog CEO API.
- Generates a slideshow of images for the selected breed.
- Uses modern JavaScript features: **Fetch API**, **Promises**, and **Async/Await**.
- Includes user-friendly features: loading indicator, reset button, hover effects.
